# Atas de Reunião

## Guia para novos registros
Para manter a padronização da documentação, utilize o componente abaixo:

!!! abstract "Estrutura de Registro (Template)"
    1. Crie um arquivo `.md` nomeado como `Ata_DD_MM_AAAA.md`
    2. Copie o bloco de código abaixo e preencha os campos.
    3. Abra o arquivo `mkdocs.yml` na raiz do projeto e adicione a nova ata na seção `nav`
